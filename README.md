🚦 Wayzen – Smart Safe Route Navigation System



Wayzen is a smart route navigation web application that helps users find safer travel paths based on safety-related data.



---



\## 🌟 Features



\- 📍 Interactive map interface

\- 🛣️ Route visualization

\- ⚠️ Safety-based route analysis

\- 🧠 Backend processing using Python

\- 💾 Local database integration

\- 🎨 Clean and responsive frontend UI



---



\## 🛠️ Tech Stack



Frontend:

\- HTML

\- CSS

\- JavaScript



Backend:

\- Python (Flask)



Database:

\- SQLite
